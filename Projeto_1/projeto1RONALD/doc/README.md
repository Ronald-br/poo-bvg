# Sistema de Gestão de Clientes

Este é um sistema simples de gestão de clientes para uma loja. A implementação foi feita em Python, utilizando os conceitos de **Programação Orientada a Objetos (POO)** para refatorar um código estruturado.

## Objetivo

O objetivo deste projeto é criar uma aplicação que armazene e manipule dados de clientes de uma loja. Utilizando a Programação Orientada a Objetos, a classe `Cliente` foi criada para encapsular as informações e fornecer métodos para exibir e atualizar os dados do cliente de forma modular e escalável.

## Funcionalidades

- **Armazenamento de informações do cliente**: Nome, idade e saldo do cliente.
- **Exibição das informações**: Exibe os dados do cliente (nome, idade, saldo).
- **Atualização de saldo**: Permite atualizar o saldo do cliente.
