## Descrição
Implementação do sistema de seguros de carro com as classes:
- `Cliente` (nome, cpf)
- `Modelo` (nome)
- `Carro` (ano, marca, modelo, cor, placa)
- `SeguroVeiculo` (base)
- `SeguroCarro` / `SeguroMoto` (herdam de SeguroVeiculo)

## Estrutura
- `src/` : código fonte
- `docs/` : documentação e UML
- `tests/` : testes simples
